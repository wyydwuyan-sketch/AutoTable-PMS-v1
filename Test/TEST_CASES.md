# UI 自动化主流程用例清单

## 范围
- 目标系统：AutoTable 前端 + 后端
- 测试类型：UI 自动化（Playwright）
- 用例目录：`Test/ui/*.module.spec.ts`

## 模块与用例

### 模块：认证（`auth.module.spec.ts`）
- TC-AUTH-001 登录页显示账号密码输入与提交按钮  
前置条件：服务可用  
步骤：访问 `/login`  
期望：账号输入框、密码输入框、登录按钮可见，密码框为 `password` 类型
- TC-AUTH-002 管理员登录后进入主表格页  
前置条件：默认账号可登录（`admin/admin123`）  
步骤：输入账号密码并点击“登录”  
期望：跳转到 `/b/base_1/t/*/v/*`，主表格及“筛选/新增记录”按钮可见

### 模块：记录（`records.module.spec.ts`）
- TC-RECORD-001 新增记录弹窗展示基础输入和保存按钮  
前置条件：已登录并进入主表格页  
步骤：点击“新增记录”  
期望：弹窗出现，至少存在一个输入框，“保存”按钮可见
- TC-RECORD-002 保存新增记录后出现成功提示  
前置条件：已登录并进入主表格页  
步骤：输入唯一名称，点击“保存”并确认  
期望：新增记录弹窗关闭，页面出现“已新增记录。”提示

### 模块：视图配置（`views.module.spec.ts`）
- TC-VIEW-001 可从侧边栏进入视图配置页并看到新增入口  
前置条件：已登录  
步骤：点击侧边栏“视图配置”  
期望：URL 为 `/config/views`，出现“新增当前表视图”按钮
- TC-VIEW-002 新增视图后自动进入初始化向导并可完成初始化  
前置条件：已登录  
步骤：新增视图并保存确认，进入初始化弹窗后点击“完成初始化”  
期望：URL 先包含 `setup=1`，完成后去除 `setup`，出现“视图初始化已完成。”提示

### 模块：视图组件配置（`components.module.spec.ts`）
- TC-COMP-001 可进入视图组件配置页并显示字段编辑入口  
前置条件：已登录  
步骤：点击侧边栏“视图组件配置”  
期望：URL 为 `/config/components`，存在“编辑组件”按钮
- TC-COMP-002 字段编辑弹窗支持配置和组件预览双 Tab  
前置条件：已登录并进入视图组件配置页  
步骤：点击任一“编辑组件”，切换到“组件预览”  
期望：弹窗中“配置/组件预览”Tab 均可见，预览卡片渲染且存在选中态

### 模块：组件参考（`showcase.module.spec.ts`）
- TC-SHOWCASE-001 组件参考页为纯浏览模式（无应用按钮）  
前置条件：已登录  
步骤：进入“组件参考”页面  
期望：标题为“字段组件参考 / Field Component Reference”，显示“页面模式：浏览模式”，且不出现“应用到目标视图/配置并应用”
- TC-SHOWCASE-002 组件参考页可从其他配置页再次进入  
前置条件：已登录  
步骤：先进入“视图组件配置”，再进入“组件参考”  
期望：成功到达 `/config/showcase` 并显示组件参考标题

### 模块：工作流（`workflow.module.spec.ts`）
- TC-WORKFLOW-001 可进入工作流配置页并展示状态配置项  
前置条件：已登录  
步骤：点击侧边栏“工作流配置”  
期望：URL 为 `/config/workflow`，显示“状态字段/终态定义/保存”控件
- TC-WORKFLOW-002 可保存工作流配置  
前置条件：已登录并进入工作流配置页  
步骤：点击“保存”  
期望：页面出现“工作流配置已保存。”提示

### 模块：看板（`kanban.module.spec.ts`）
- TC-KANBAN-001 可切换到看板视图并展示状态列  
前置条件：已登录  
步骤：点击侧边栏“看板”视图  
期望：URL 以 `/kanban` 结尾，页面显示“按状态分列，可拖拽更新状态”，并出现状态列
- TC-KANBAN-002 点击看板卡片可打开记录详情抽屉  
前置条件：已登录并进入看板视图  
步骤：点击任一卡片  
期望：出现“记录详情”抽屉

### 模块：大屏（`dashboard.module.spec.ts`）
- TC-DASH-001 可从头部按钮进入大屏页面并返回表格  
前置条件：已登录  
步骤：点击头部“大屏”按钮，然后点击“返回表格”  
期望：先进入 `/dashboard`，再返回主表格路由

### 模块：端到端主流程（`flow.module.spec.ts`）
- TC-FLOW-001 登录后可完成新增视图、配置字段与配置字段组件  
前置条件：服务可用  
步骤：登录 -> 进入视图配置并新增视图 -> 在初始化向导中选择可见字段并设置字段组件 -> 完成初始化  
期望：成功进入组件配置页并完成初始化，字段状态与组件类型（如 `textarea`）正确保存
